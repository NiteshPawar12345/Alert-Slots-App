Visa Slot Alert App

Visa Slot Alert App is a simple full-stack MERN project.
In this project, users can create, view, update and delete visa slot alerts.
The UI updates in real time, so there is no need to refresh the page again and again.

---------------------------------------------------------------------------------------------

Project Description

This application is used to manage visa slot alerts based on country and city.
Each alert contains visa type and status like Active, Booked or Expired.

The frontend of this project is built using React and Tailwind CSS.
The backend is developed using Node.js and Express.js.
MongoDB is used as the database to store visa slot data.

---------------------------------------------------------------------------------------------

Features

Users can create new visa slot alerts by entering country, city and visaType.
All visa alerts are displayed in card format.
Users can change the visa status from Active to Booked and then to Expired.
Users can also delete any visa alert.
The UI updates instantly without refreshing the page.
The project follows REST API architecture.

---------------------------------------------------------------------------------------------

Tech Stack
Frontend

React.js is used to build the user interface.
Tailwind CSS is used for styling.
Axios is used to connect frontend with backend APIs.

Backend

Node.js is used for server-side logic.
Express.js is used to create REST APIs.

Database

MongoDB is used as the database with Mongoose for schema management.
